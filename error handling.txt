GPT 파싱 에러 문제는 GPT가 응답에 추가 설명을 포함시키는 것이니, JSON만 정확하게 추출, 정규식을 사용하여 
 첫 번째 완전한 JSON 객체만 정확하게 추출, 중첩된 중괄호도 올바르게 처리

* GPT 프롬프트 요구사항:
   - Final 분석시 다이버전스 설명은 반드시 한글로 작성
   - 응답 형식:
     {
       "divergence": {
         "timeframe": "4h",
         "type": "하락",
         "description": "4시간봉: 하락 다이버전스 발생. 상승 모멘텀 약화 신호"
       }
     }
   - 불필요한 기술적 설명이나 영문 설명 제외
   - JSON 형식 엄격히 준수

   지피티 응답을 텔레그램에 알람을 할때 JSON 파싱 로직(중첩된 중괄호 처리)에 신경을 써야함.
